{{- if not .Values.builderPoolDefaults.privileged }}

── /dev/fuse via containerd base_runtime_spec ─────────────────────────
Builder pods get /dev/{fuse,kvm} injected by containerd's
`base_runtime_spec` (OCI `linux.devices` — runc mknods inside the
container's /dev). No device plugin runs; no extended resource is
requested. kvm pods route to .metal via nodeSelector.

Prerequisite:
  • Node containerd configured with `base_runtime_spec` declaring
    /dev/{fuse,kvm} (NixOS AMI: nix/base-runtime-spec.nix; k3s VM
    fixture: services.k3s.containerdConfigTemplate).

Verify on a builder pod:
  kubectl -n {{ .Values.namespaces.builders.name }} debug rio-builder-x86-64-0 -it --image=busybox -- ls -l /dev/fuse
{{- end }}

{{- if not .Values.builderPoolDefaults.privileged }}

── Seccomp Localhost profile (node-baked, ADR-021) ────────────────────
Profiles are baked into the node image and copied to
/var/lib/kubelet/seccomp/operator/ by systemd-tmpfiles BEFORE kubelet
starts (nix/nixos-node/hardening.nix on EKS; fixtures/k3s-full.nix in
VM tests). Builder pods use seccompProfile: Localhost (RuntimeDefault +
pivot_root − ptrace/bpf/setns/process_vm_*).

Verify on a builder node:
  kubectl debug node/$(kubectl get node -l rio.build/node-role=builder -o name | head -1 | cut -d/ -f2) -it --image=busybox -- ls -l /host/var/lib/kubelet/seccomp/operator/

Profile updates: change nix/nixos-node/seccomp/rio-*.json → rebuild AMI
(`cargo xtask k8s -p eks up --ami`) → Karpenter Drift rolls nodes on the
new amiSelectorTerms tag. See docs/src/security.md
r[worker.seccomp.localhost-profile].
{{- end }}

{{- if .Values.builderPoolDefaults.privileged }}

── privileged:true escape hatch ───────────────────────────────────────
WARNING: builderPoolDefaults.privileged=true is active. This disables seccomp
entirely and is UNACCEPTABLE for multi-tenant deployments (ADR-012).
It is intended for clusters whose containerd lacks `base_runtime_spec`
device injection or idmap-mount support. Production on EKS/GKE should
use privileged:false (the chart default).
{{- end }}
