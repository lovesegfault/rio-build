# Phase 3: Kubernetes Native (Months 12-17)

Phase 3 is split into two sub-phases to manage scope:

- **[Phase 3a](./phase3a.md)** (Months 12-15): K8s operator, CRDs, FUSE worker stores at scale, deployment manifests
- **[Phase 3b](./phase3b.md)** (Months 15-17): Production hardening --- RBAC, NetworkPolicy, PDB, EKS-specific configs, K8s-aware retry, basic GC, FOD proxy

See the individual sub-phase docs for detailed task lists and milestones.
